## Para usar a aplicação:

### Abre a aplicão em ambiente de desenvolvimento

```bash
git clone https://github.com/rodrigokovalski/teste-docket.git
$ cd teste-docket
$ yarn
$ yarn dev
```

### Manter processo ativo para o uso da aplicação

```bash
$ yarn server
```
